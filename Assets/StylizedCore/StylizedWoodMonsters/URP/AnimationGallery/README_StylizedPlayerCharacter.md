# 🧍 Stylized Player Character (Free)

## 📌 Asset Overview

Stylized humanoid character, fully rigged and animated, ready for games/cinematics/real-time projects.

## 📦 Included

* 1 model: `Stylized Player Character`
* 3 animations (In-Place, no Root Motion)
* PBR textures in 2K and 4K
* 1 texture set in 2K + 1 texture set in 4K
* 1 material
* Optional `AnimationGallery` demo viewer

## 📊 Model Stats

* Triangles: `30,682`

## 🎬 Animations (3)

`Idle`
`Walking`
`Running`

## 🎨 Textures & Rendering

Texture maps included:
- Albedo (+Transparency)
- Normal
- Ambient Occlusion
- Height
- Metallic + Smoothness
- Mask Map (HDRP)
- Emission

## ⚙️ Compatibility & Notes

* `AnimationGallery` created in Unity 6 (`6000.3.8f1`)
* Compatible with Built-in, URP and HDRP
* Humanoid rig compatible with Mixamo
* Designed/tested in Unity 6 (`6000.3.8f1`). Other versions may require minor setup
* Models/textures can be exported to other engines, but setup/compatibility is not included

### Input System (Demo Scene)

Demo uses Built-in Input Manager. If using New Input System:

1. `Edit > Project Settings > Player`
2. `Other Settings > Active Input Handling`
3. Set to `Both`
